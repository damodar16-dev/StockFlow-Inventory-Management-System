# 📦 StockSphere

StockSphere is an Inventory Management System developed using Spring Boot.

## 🚀 Features

- User Login Authentication
- Role Based Access (ADMIN / USER)
- Dashboard
- Add Product
- Edit Product
- Delete Product
- Search Products
- Category Filter
- Sort by Name
- Sort by Price
- Pagination
- Low Stock Alert
- Recent Products
- Inventory Value
- PDF Export
- Form Validation

## 🛠 Technologies Used

- Java 21
- Spring Boot
- Spring MVC
- Spring Data JPA
- Spring Security
- Thymeleaf
- MySQL
- Bootstrap 5
- Maven

## 💾 Database

MySQL Database

Database Name:

```
inventory
```

## ▶️ Run Project

1. Clone Repository

```
git clone <repository-link>
```

2. Open in IntelliJ / STS

3. Create MySQL database

```
CREATE DATABASE inventory;
```

4. Update application.properties

5. Run the project
